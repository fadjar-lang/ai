aplikasi ini adalah aplikasi untuk perbandingan atau compare spesifikasi laptop menggunakan 2 model ai
- pertama kita input 2 laptop yang berbeda pada halaman input
- setelah itu kita submit
- setelah di submit maka akan muncul perbandingan spesifikasi dari 2 model ai tersebut

manfaat dari aplikasi ini adalah kita bisa membandingkan spesifikasi laptop yang kita inginkan
